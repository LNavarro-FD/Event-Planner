Quick start for a new event
1
Add suppliers
Suppliers → add each company, set type, paste OneDrive link. Click "Request docs" to send compliance emails.
2
Draw site plan
Site Planner → choose event → enter dimensions → place items from library. Saves and syncs automatically.
3
Work through To-Do
To-Do → lifecycle overview shows progress by category. Click 📋 on any task to add quotes or design brief.
4
Staff the event
Staff → add people, assign roles, set shifts. Default start = event time minus 1 hour. Print rotas from the app.
5
Plan print & design
Print → mark items Required / Maybe / Not required. Fill in quotes and specs. Print procurement sheet.
6
Run build week
Kanban → Build Week. Mon 10 → Sat 15 ★ → Mon 17 Aug. Click any cell to assign. Drag chips to move.
💡 All data saves to Firestore automatically — open on any device and it picks up where you left off. Every module has a Print button for physical copies.
